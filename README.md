# Justice AI – Legal Document Summarization System

Justice AI is an AI-powered web application that summarizes lengthy legal documents
into concise, human-readable summaries using transformer-based NLP models.

## Features
- Upload PDF or TXT legal documents
- Abstractive summarization using NLP
- Clean and simple web interface

## Tech Stack
- Python, Flask
- NLP (Transformers)
- HTML, CSS, JavaScript

## Impact
- Reduced legal document reading time by ~70–80%
- Helps non-technical users understand complex legal text

## How to Run
1. Install dependencies
2. Run backend.py
3. Open index.html
